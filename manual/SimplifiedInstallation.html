Упрощенная установка

Вы можете использовать эту процедуру, если вы загрузили официальную предварительно скомпилированную версию Flora-2; ее нельзя использовать, если вы получили программное обеспечение из репозитория SVN. Преимущество этой процедуры в том, что она не требует специального программного обеспечения, такого как компилятор C++ или make/nmake для установки. Отсутствие требований этого дополнительного программного обеспечения особенно выгодно пользователям Windows и, в некоторой степени, также пользователям Mac.

Официальный релиз также включает одну из последних версий XSB, совместимую с версией Flora-2. Таким образом, нет необходимости загружать XSB отдельно.

Unix-системы (Linux, Mac, BSD и т.д.). Загрузка в стиле Unix - это файл flora2.run. Это самораспаковывающийся архив. Поместите его в соответствующий каталог установки DIR, а затем выполните его:

sh ./flora2.run

Вышеупомянутая команда создаст папку Flora-2 с двумя подпапками: flora2 и XSB. Скрипт для запуска Flora-2 - DIR / Flora-2 / flora2 / runflora.

Windows Windows-установщик называется flora2.exe. Поместите его на рабочий стол и дважды щелкните по нему. Вам будет задан обычный ряд вопросов для установщика Windows. В итоге, как XSB, так и Flora-2 будут установлены с соответствующими ярлыками, размещенными на рабочем столе и в меню «Пуск».

Для 64-битных машин будет автоматически установлена ​​64-разрядная версия. В противном случае будет установлена ​​32-разрядная версия.

Запуск Flora-2 . Команда запуска Flora-2 - ./runflora (Linux / Mac / BSD / etc.) Или runflora.bat (Windows). Эти скрипты находятся во вложенной папке flora2 установочной папки.

Примечание : вы можете заметить, что во время нескольких первых использований системы под Windows некоторые внутренние файлы Flora-2 могут быть перекомпилированы самостоятельно. Это нормально. Причиной является некорректная работа Windows, при которой она часто меняет временные метки файлов во время распаковки.
